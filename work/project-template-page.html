creating a reusable project page template for visual consistency

ui/ux designer & html integrator

|

15-22.5 hours

i created a reusable html template for exo's project pages to ensure visual consistencty across all projects. this templates standardizes the style and structure of new project pages, making it easier and faster for new project pages to be integrated.

the problem

i wanted the style and structure of exo's project pages to be consistent across all of them. this would make it easier for new project pages to be integrated, and would also create a more cohesive experience for users browsing through exo's projects.

examples of exo's project pages with different styles and structures

my process

after delopying the navilens project page, i decided that creating a reusable template would be a good idea. i sed the navilens project page as a base and added other components to it to create a comprehensive template that could be used for all future projects.

navilens project page used as a base for the project template page

figma - prototype of the project template page

key deliverables

the key deliverable for this project was the final html template page that could be reused for all projects. this template includes sections for the project description, goals, process, progress, faq, and important dates.

codepen - html template for the project template page

the solution

all new project pages now have the same structure and style. there is a template that we can refer to when creating a new page, which makes the design process faster and the style is already defined. whem implementing the pages, we can just use the template and add or remove components as needed. this makes the implementatioin process easier and faster.

reflections & learnings

i learned that consistency is crucial on a website, especially when multiple people are working on it. by defining a style and structure, we can ensure that there are no additional styles or components that need to be created. this also avoids confusion and makes the design and implementation process faster.
